# Deep RL Portfolio Optimization

Research project for deep reinforcement learning methods applied to constrained
resource allocation with a focus on dynamic portfolio optimization.

## Local Setup

1. Create or refresh the local virtual environment:
   `python3 -m venv .venv`
2. Activate it:
   `source .venv/bin/activate`

## Status

Repository bootstrap complete. Modeling code, experiment structure, and
dependency choices are intentionally left minimal until the first concrete
implementation step.
