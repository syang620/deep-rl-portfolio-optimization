# Deep RL Portfolio Optimization

Research project for deep reinforcement learning methods applied to constrained
resource allocation with a focus on dynamic portfolio optimization.

## Local Setup

1. Create the Conda environment:
   `conda create -n drl-portfolio-opt python=3.11`
2. Activate it:
   `conda activate drl-portfolio-opt`
3. If activation is not available in the current shell, initialize Conda for
   `zsh` first:
   `conda init zsh`
